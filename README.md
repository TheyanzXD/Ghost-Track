### Instalation on Linux (deb)
```
sudo apt-get install git
sudo apt-get install python3
```

### Instalation on Termux
```
pkg install git
pkg install python3
```

### Usage Tool
```
git clone https://github.com/TheyanzXD/Ghost-Track
cd Ghost-Track
python3 GTrack.py
```
